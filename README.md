# jogo-da-forca-tibiano
Um algoritmo contendo um jogo da forca na versão tibiana! Contendo +150 palavras relacionadas ao jogo tibia, onde você pode colocar seu nome antes de começar, caso acerte, vai mostrar a quantia de acertos, e caso não consiga acertar a palavra em até 6 tentativas, irá mostrar o boneco pendurado e qual seria a palavra correta!
